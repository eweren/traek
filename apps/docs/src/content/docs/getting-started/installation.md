# Installation

## Requirements

- Svelte 5 or later
- SvelteKit 2 or later (recommended)
- Node.js 20+

## Install

```bash
npm install @traek/sdk
# or
pnpm add @traek/sdk
# or
yarn add @traek/sdk
```

## Peer Dependencies

Træk requires Svelte 5 as a peer dependency:

```bash
npm install svelte@^5
```
