## Full token reference

The complete machine-readable token file is at `packages/svelte/src/lib/theme/tokens.json`. It follows the [W3C DTCG format](https://tr.designtokens.org/format/) and can be consumed directly by Style Dictionary or Figma Tokens.

See the [Design Tokens & Figma Sync](/guides/design-tokens) guide to set up automated token export and Figma Variable sync.
