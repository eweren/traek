# Træk Community Events & Conference Strategy

> Last updated: 2026-03-08

## Overview

This document outlines Træk's full community events program: conference presence, hackathon sponsorships, virtual office hours, and the local meetup program. The goal is to establish Træk as the recognized library for spatial AI conversation UIs by meeting developers where they already gather.

## Hackathon Sponsorship Strategy

### Why Hackathons

Hackathons compress the adoption funnel: developers encounter a problem, discover a tool, build with it, and share the result — all in 48 hours. A $2k sponsorship at a well-targeted AI hackathon can generate more working demos than months of blog posts.

### Target Hackathon Types

1. **AI-native hackathons** — Focus on LLM/agent workflows. Træk fits naturally as the UI layer.
2. **Svelte jams** — SvelteHack and similar community jams. Directly in our ecosystem.
3. **Open-source sprints** — MLH Open Source Weekends, GitHub Game-Off (for lateral exposure).
4. **University AI hackathons** — Early adopter pipeline.

### Specific Targets

| Hackathon | Tier | Sponsorship Level | Activation |
|-----------|------|-------------------|------------|
| **SvelteHack** (annual) | Primary | $1,000–2,500 | Named challenge + prize |
| **MLH Local Hack Day** | Primary | $500–1,000 | Workshop slot |
| **Lablab.ai Hackathons** (monthly) | Primary | $500 | Træk challenge track |
| **HuggingFace Spaces Hackathon** | Secondary | Workshop only | Demo + tutorial |
| **TreeHacks (Stanford)** | Secondary | $2,000 | AI UI track sponsorship |
| **PennApps** | Secondary | $1,500 | Best AI Interface prize |
| **HackMIT** | Exploratory | $2,000 | TBD based on track |

### Sponsorship Activation Checklist

For each sponsored hackathon:

- [ ] Create a Træk-specific challenge prompt (e.g. "Build an AI tutoring tool using Træk's branching canvas")
- [ ] Prepare 30-minute intro workshop ("Træk in 30 minutes")
- [ ] Provide participants with pre-scaffolded starter template
- [ ] Set up a #traek channel in hackathon Discord
- [ ] Assign a Træk team member to be present (in-person or async-online)
- [ ] Offer $500 prize for "Best Træk Integration" or equivalent named award
- [ ] Collect all demos for the Træk showcase page post-event

### Annual Hackathon Budget Estimate

| Category | Amount |
|----------|--------|
| Primary hackathons (3–4) | $6,000–8,000 |
| Secondary hackathons (3–4) | $4,000–6,000 |

## Monthly Virtual Office Hours

### Format

**Name:** Træk Open Hours
**Cadence:** Last Thursday of every month, 11:00 AM Pacific / 19:00 UTC
**Duration:** 60 minutes
**Platform:** Discord Stage (public, recorded → YouTube)

### Agenda Template

```
00:00 – 00:05  Welcome & intro (host)
00:05 – 00:20  Demo: new feature or community project (rotating)
00:20 – 00:40  Open Q&A — submit questions in #office-hours channel in advance
00:40 – 00:55  "Build with me" — live-code a small Træk feature or integration
00:55 – 01:00  Wrap-up, next month preview, CTA
```

### Host Rotation

- Month 1: Core team (introduce, set tone)
- Month 2: Community contributor demo
- Month 3: Guest speaker (partner SDK, AI provider)
- Repeat cycle

### Promotion Checklist (1 week before)

- [ ] Post announcement in #announcements on Discord
- [ ] Tweet/post to social with topic teaser
- [ ] Add to Træk newsletter footer
- [ ] Create calendar invite link (Luma / lu.ma preferred)
- [ ] Open #office-hours for question submissions

### Post-Event Actions

- [ ] Upload recording to YouTube (unlisted → listed after 24h review)
- [ ] Write 300-word summary for blog/newsletter
- [ ] Capture any GitHub issues raised during session
- [ ] Thank guest speaker in Discord and on social
